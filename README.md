# Review-Analyzer-from-Product-Reviews  

Algorithms used:  
1. Perceptron Algorithm  
2. Average Perceptron Algorithm  
3. Pegasos Algorithm  

Data:  
reviews_train.tsv : 4000 examples  
reviews_validation.tsv : 500 examples  
reviews_test.tsv : 500 examples  
Each dataset has reviews with label (+1 or -1) associated with them.  

Translating reviews to feature vectors:  
Unigram model is being used for that purpose.
